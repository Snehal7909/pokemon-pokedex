# Pokemon Search Engine

A full-stack Pokemon search application built using React and Node.js.

## Features
- Search Pokemon by name
- REST API backend
- Caching using LRU Cache
- Responsive UI

## Tech Stack

Frontend:
- React
- Axios

Backend:
- Node.js
- Express.js

## Run Project

### Frontend
cd frontend
npm install
npm run dev

### Backend
cd backend
npm install
node server.js
